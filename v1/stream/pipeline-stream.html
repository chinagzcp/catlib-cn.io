管道流

管道流允许同步化的形式在两个不同线程的传递数据。

构建管道流

函数原型
参数 描述
capacity 管道流的容量
sleep 等待数据时线程sleep的时间

OnRead

当读取时触发的事件,传入参数为管道流实例。

函数原型

CanRead

判断管道流是否可以被读取。

函数原型

CanWrite

判断管道流是否可以被写入。

函数原型

Position

获取管道流实际传输的位置(流量)（已被读取）。

函数原型

Length

获取管道流需要传输的总长度。如果为0则表示没有限制。

函数原型

CanSeek

是否能够设定Position的位置，在管道流中这个值始终返回false。

函数原型

Closed

判断管道流是否已经被关闭。

函数原型

SetLength

设定管道流期望传输的总长度（预期长度）。

函数原型
参数 描述
value 预期需要传输的长度

Read

读取管道流中的数据,返回值为实际读取的长度。

函数原型
参数 描述
buffer 读取的缓冲区
offset 读取缓冲区起始偏移量
count 期望读取的长度

Write

向管道流中写入数据,如果写入的数据不能完成写入，那么会挂起等待写入完成。

函数原型
参数 描述
buffer 写入的缓冲区
offset 写入缓冲区起始偏移量
count 期望写入的长度

Close

关闭管道流,关闭管道流后不能够再写入数据，但是依旧可以读取完全部数据。

函数原型

Dispose

释放管道流，释放后将不能够读取和写入。
